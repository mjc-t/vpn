+ Dynamic Routing
- 터널주소 광고
- 물리적 주소 광고 포함 안함

1. ISAKMP 정책 설정
crypto isakmp policy 10
     authentication pre-share
2. Peer 인증
crypto isakmp key 암호 address 피어주소
3. 보호대상 트래픽 지정
access-list 100 permit gre host 출발지주소 host 목적지주소
4. IPsec 정책 설정
crypto ipsec transform-set 정책이름 esp-aes esp-md5-hmac
5. Crypto Map 설정
crypto map 크립토이름 10 ipsec-isakmp
     match address 100
     set peer 피어주소
     set transform-set 정책이름
6. Crypto Map 적용
int ...
     crypto map 크립토맵이름
